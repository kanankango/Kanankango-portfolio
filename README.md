## ⚡ Running the Project

Clone the repository:

```id="clone1"
git clone https://github.com/kanankango/kanankango-portfolio.git
```

Go to project folder:

```id="cd1"
cd kanankango-portfolio
```

Install dependencies:

```id="install1"
npm install
```

Run development server:

```id="run1"
npm run dev
```
